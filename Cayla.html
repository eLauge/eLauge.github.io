CAYLA

Computer Aided Yield Line Analysis

My BSc thesis in Building Design at Aarhus University was about automating the yield line theory. After the submission I started to implement the findings of the thesis as a Grasshopper plugin. The result was CAYLA, a plugin which automatically calculates the bending moment in plates based on a yield line pattern drawn by the user. The plugin can automatically create the documentation as a Microsoft Word format.

Features:

  • Supports with arbitrary fixity
  • Free edges
  • Surface loads
  • Line loads
  • Column supports
  • Holding down forces
  • Automatic calculation documentation
