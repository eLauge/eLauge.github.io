Manatee

During the course Form-finding I choose to write my assignment about dynamic relaxation. The result was the Grasshopper plugin Manatee. The plugin is an implementation of the Position-based dynamic relaxation algorithm much like Kangaroo 2.

Features:

  • Non-linear bar elements
  • Finite difference rod elements
  • Point loads
  • Line loads
  • Points anchors
